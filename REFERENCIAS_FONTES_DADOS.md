# Fontes de Dados e Referências Eletrônicas

Este documento lista as fontes oficiais e os endereços eletrônicos (links) utilizados para a coleta de dados primários referentes à Avaliação P1 de Finanças Públicas.

---

## 🌏 1. Dados Macroeconômicos (Indonésia)

Os dados referentes ao Governo Central da Indonésia foram extraídos de bases de dados internacionais de referência (Banco Mundial e FMI).

* **Produto Interno Bruto (PIB Nominal):** [World Bank - GDP (current US$) - Indonesia](https://data.worldbank.org/indicator/NY.GDP.MKTP.CD?locations=ID)
* **Índice de Inflação (CPI):** [World Bank - Consumer Price Index - Indonesia](https://data.worldbank.org/indicator/FP.CPI.TOTL?locations=GH-ID)
* **Receita e Despesa Total do Governo Geral:** [IMF - World Economic Outlook Database (April 2024)](https://www.imf.org/en/publications/weo/weo-database/2024/april/weo-report?c=536&s=GGR,GGX&sy=2013&ey=2025&ssm=0&scsm=1&ssd=1&scc=1&ssc=1&sic=0&sort=country&ds=.&br=0)

---

## 🏛️ 2. Dados Subnacionais (Betim e Governador Valadares)

Para a análise dos municípios mineiros, foram utilizadas as bases de dados do Tesouro Nacional, IBGE e portais de indicadores sociais.

* **Contabilidade Fiscal (Receita, Despesa e População):** [Siconfi/STN - Consulta Finbra (Dados Contábeis de Municípios)](https://siconfi.tesouro.gov.br/siconfi/pages/public/consulta_finbra/finbra_list.jsf)
* **Índice de Preços (IPCA):** [Ipeadata - Séries Históricas de Inflação](https://ipeadata.gov.br/Default.aspx)
* **Produto Interno Bruto (PIB Municipal):** [IBGE Sidra - Tabela 5938 (PIB a preços correntes)](https://sidra.ibge.gov.br/Tabela/5938)
* **Indicadores Sociais (IDEB):** * [Portal QEdu - Especial Minas Gerais (Indicadores)](https://qedu.org.br/especial/minas-gerais/?indicador=ideb&view=mapa&dependencia=2&ciclo=AI)
    * [Portal QEdu - Perfil do IDEB por Município](https://qedu.org.br/uf/31-undefined/ideb)

---

## 🛠️ Notas Metodológicas
- Todos os dados foram coletados para o período de **2014 a 2023**.
- Os valores financeiros extraídos destas fontes passaram por processo de deflação conforme detalhado no documento `Metodologia_Deflacao.md`.
